## ✨ Features

🚀 Fitur Utama

Aplikasi ini menyediakan berbagai fitur lengkap sesuai kebutuhan proyek:

🔐 1. Autentikasi Pengguna

Login menggunakan Email & Password

Login melalui Google (Google Sign-In)

Registrasi akun baru

Fitur Lupa Password & Ganti Password

📚 2. Eksplorasi Anime

Menampilkan daftar anime dengan sistem pagination / load more

Pencarian anime berdasarkan judul

Filter anime berdasarkan genre

⭐ 3. Daftar Favorit

Menambahkan anime ke daftar favorit

Menghapus anime dari daftar favorit

Data favorit tersimpan dan tersinkronisasi real-time melalui Firestore

👤 4. Profil Pengguna

Menampilkan informasi akun pengguna

Fitur logout untuk keluar dari aplikasi

##🔒 Keamanan

Aturan Firestore Production: Akses data dibatasi per pengguna (user-scoped) dan hanya dapat diakses setelah autentikasi.

Autentikasi Aman: Menggunakan Firebase Authentication dengan dukungan Login Google (Google Sign-In).

Penandatanganan Rilis: Aplikasi ditandatangani menggunakan production keystore sehingga siap untuk deployment di Play Store.

## 🚀 Getting Started

### Prerequisites

Sebelum menjalankan proyek ini, pastikan Anda telah menyiapkan hal-hal berikut:

Flutter SDK: versi 3.38.1 atau lebih tinggi

Dart SDK: versi 3.10.0 atau lebih tinggi

Android Studio atau VS Code dengan ekstensi Flutter

Java JDK 17: diperlukan untuk proses build Android

Akun Firebase: digunakan untuk autentikasi dan database
